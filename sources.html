무료종목 시그널 data/stocks.json · data/sd/ · data/home_reco.json

NDX·SPX 512종목의 가격·테크니컬 22지표·EPS(트레일링/포워드)는 yfinance, 공매도잔량·커버일수는 FINRA(격주 공시라 가격 기준일보다 오래됨 — 페이지에 별도 기준일 표기). 구성종목 리스트(data/members.json)는 사내 DB index_constituents의 최신 스냅샷입니다 — 지수 구성·회사명·GICS 섹터는 공개 정보로 재구성 가능한 수준이며, 지수 정기변경 시 재생성해 갱신합니다. 갱신 잡에는 완전성 게이트가 있어 데이터 소스 부분 장애 시 갱신을 중단하고 이전본을 유지합니다.

기준일—공매도 기준일—구성종목 스냅샷—갱신매일 08:05 + 08:40 KST (GitHub Actions)

무료시장 국면 data/regime.json

매크로 39지표(성장·노동·물가·금융·금리·주택)는 FRED(세인트루이스 연준 공개 API), 레짐별 자산·섹터·팩터 조건부 성과는 yfinance. 각 지표의 '서프라이즈'는 자체 6개월 추세 대비 이탈이며 애널리스트 컨센서스가 아닙니다.

기준일—갱신매일 08:15 + 08:50 KST (GitHub Actions)

무료오늘의 로테이션 data/rotation_pool.json

전략 풀은 공개 논문·기관 리서치 웹 리서치로 수집·검증한 카드이며, 카드마다 원 출처 링크를 표기합니다. 성과 수치는 원 출처 인용(미검증)이고, 랩이 자체 검증한 전략에는 별도 판정 배지가 붙습니다. 기준일·갱신일은 로테이션 페이지 상단에 표기됩니다.

무료현재 포트폴리오 구성 data/strategy_holdings.json

전략별 현재 보유 종목·자산과 비중을 공개합니다. 자산배분 전략(크로스에셋 RP·섹터 RP·변동성 관리)은 yfinance 가격으로, 종목선택 전략(EPS 리비전·단일팩터 top10·팩터 모멘텀)은 사내 DB 신호로 산출합니다. 리서치 재현 기준의 구성이며 실제 운용 계좌·실시간 비중이 아닙니다.

사내 DB전략 백테스트 · 판정 data/strategy_backtests.json · data/strategy_detail.json

전략 탐색기의 백테스트 곡선·지표·판정은 사내 리서치 DB(FactSet 라이선스 데이터: 가격·애널리스트 추정치·팩터)에서 파생된 결과입니다. 공개 범위는 집계 성과(월간 누적수익·낙폭·연환산 지표)와 정성 판정으로 한정하며, 원천 수치(가격·추정치·팩터값)와 종목 선정 결과(보유 리스트)는 공개하지 않습니다. 백테스트 구간은 전략별로 다르며 탐색기 각 패널에 표기됩니다. 무비용·월말(gross) 기준.
